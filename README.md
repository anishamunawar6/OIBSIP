# BMI Calculator

## Project Overview
BMI Calculator is a Python-based GUI application that calculates Body Mass Index (BMI) and classifies it into different health categories.

## Objective
The objective of this project is to create a user-friendly BMI calculator with data storage and visualization features.

## Features
- User-friendly GUI using Tkinter
- Calculate BMI using weight and height
- BMI category classification:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Input validation
- Color-coded results
- Store BMI records using SQLite database
- View previous BMI history
- Display BMI trend graph using Matplotlib

## Technologies Used
- Python
- Tkinter
- SQLite
- Matplotlib
